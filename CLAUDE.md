# Wedding Invitation — Eko & Melati

## Stack
Static HTML/CSS/JS — GitHub Pages. No framework.

## Key Sections
- **Opening cover** (invite mode only, `?to=Nama`): overlay before clicking Buka Undangan
- **Hero** (`#hero`): Bismillah → Assalamu'alaikum → Yth. → invitation → names → date
- **Couple** (`#couple`): groom + bride with fullname & parents description
- **Closing** (`#closing`): closing text → hormat → names → parents → Wassalam → footer

## UI Patterns
- i18n: `data-i18n` attributes + `I18N` object
- Guest name from URL params `?to=X&title=Y`
